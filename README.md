# galer-a-administrable
Este sistema web gestiona la carga y publicación segura de fotos/videos en "EVENTOS".  Roles: Colaboradores (carga pendiente) y Editores/Admins (aprueban, editan descripción/archivos).  Eficiencia: Paginación rápida y barra de progreso AJAX para subidas grandes. Garantiza que solo el material revisado sea público.
